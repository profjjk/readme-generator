## Project Summary
This is an application that utilizes node.js to create a professional README file from the command line. When the application is launched it prompts the user for information that will then be formatted and passed along to a newly created README file.

## Technologies Used
* [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
* [Node.js](https://nodejs.org/en/)
* [Inquirer.js](https://www.npmjs.com/package/inquirer)
