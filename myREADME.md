## Description
Dynamically generate professional quality README files from the command line using node.js and inquirer.js. The application will prompt the user to fill in different sections of the README file and then create then file when finished.

## Installation Instructions
Download the files from GitHub. Install node.js on your computer. Run ```npm init -y``` then ```npm install inquirer``` to install the necessary node modules from inquirer.js.

## Usage Information
Run command ```node index.js``` start application.
